前端►移动

移动前端开发框架选择

 

我的建议是　Bootstrap + jQuery Mobile + PhoneGap

Sencha 系列是收费的，且过于重量级，适合本地调用，比如被app封装到应用里面；

jquery系列是主流 ，开源免费、人才多、学习成本低。

Sencha Touch、jQuery Mobile、Foundation 这些前端框架负责界面和效果的编写，PhoneGap、Cordova 框架则允许通过 JavaScript 和其他接口对手机设备的调用，开创了新的移动端开发模式，真正实现了移动端的跨平台开发。
